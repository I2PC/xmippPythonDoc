67 class XmippProtReconstructHighRes(ProtRefine3D, HelicalFinder):
68  """This is a 3D refinement protocol whose main input is a volume and a set of particles.
69  The set of particles has to be at full size (the finer sampling rate available), but
70  the rest of inputs (reference volume and masks) can be at any downsampling factor.
71  The protocol scales the input images and volumes to a reasonable size depending on
72  the resolution of the previous iteration.
73 
74  The protocol works with any input volume, whichever its resolution, as long as it
75  is a reasonable initial volume for the set of particles. The protocol does not
76  resolve the heterogeneous problem (it assumes an homogeneous population),
77  although it is somewhat tolerant through the use of particle weights in the
78  reconstruction process.
79 
80  It is recommended to perform several global alignment iterations before entering
81  into the local iterations. The switch from global to local should be performed when
82  a substantial percentage of the particles do not move from one iteration to the next.
83 
84  The algorithm reports the cross correlation (global alignment) or cost (local) function
85  per defocus group, so that we can see which was the percentile of each particle in its
86  defocus group. You may want to perform iterations one by one, and remove from one
87  iteration to the next, those particles that worse fit the model."""
