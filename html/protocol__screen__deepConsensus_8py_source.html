63 class XmippProtScreenDeepConsensus(ProtParticlePicking, XmippProtocol):
64  """ Protocol to compute a smart consensus between different particle picking
65  algorithms. The protocol takes several Sets of Coordinates calculated
66  by different programs and/or different parameter settings. Let's say:
67  we consider N independent pickings. Then, a neural network is trained
68  using different subset of picked and not picked cooridantes. Finally,
69  a coordinate is considered to be a correct particle according to the
70  neural network predictions.
71  In streaming, the network is trained and used to predict in batches.
72  The network is trained until the number of particles set is reached,
73  meanwhile, a preliminary output is generated. Once the threshold is reached,
74  the final output is produced by batches.
