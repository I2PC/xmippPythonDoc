41 class XmippProtLocalCTF(ProtAnalysis3D):
42  """Compares a set of particles with the corresponding projections of a reference volume.
43  The set of particles must have a 3D angular assignment.
44  This protocol refines the CTF, computing local defocus change.
45  The maximun allowed defocus is a parameter introduced by the user (advanced).
46  The protocol gives back the input set of particles with the refine local defocus and the defocus change with relation to the global defocus."""
47  _label = 'estimate local defocus'
48  _lastUpdateVersion = VERSION_2_0
